Sistema de Fluxo de Caixa Prospera

Visão Geral do Sistema

O Sistema de Fluxo de Caixa Prospera é uma solução web (Google Apps Script) que:

  • Cria automaticamente planilhas mensais no Google Drive
  • Oferece interface web responsiva para lançamentos
  • Sincroniza dados entre dispositivos
  • Aplica identidade visual Prospera
  • Organiza movimentações por mês automaticamente
Guia de Uso (Ponta) Guia Técnico (Implementação)

GUIA DE USO — PONTA

1. Acesso

Acesse o sistema pelo link informado pela equipe:

[URL_DA_APLICAÇÃO]

2. Primeiro acesso

  1. Abra o link.
  2. Digite o nome do seu negócio.
  3. Clique em “Criar Planilha no Google Drive”.
  4. Autorize o acesso ao Google Drive, quando solicitado.

3. Como lançar

  • Use “Adicionar Receita” para vendas/entradas.
  • Use “Adicionar Despesa” para gastos.
  • O sistema calcula o saldo automaticamente.
  • Para ver tudo em detalhes, clique em “Abrir Planilha no Google Drive”.

4. Dúvidas rápidas

  • Não abriu? Verifique a conexão e tente novamente.
  • Sem acesso? Faça login na sua conta Google.
  • Planilha não aparece? Recarregue e confirme no Google Drive.

GUIA TÉCNICO — IMPLEMENTAÇÃO

1. Acessar o Google Apps Script

  1. Acesse: https://script.google.com
  2. Faça login com a conta proprietária do sistema.
  3. Clique em + Novo projeto.

2. Criar arquivos do projeto

  1. Code.gs (arquivo inicial)
    • Se já existir Código.gs ou Code.gs: abra o arquivo, pressione Ctrl+A/Cmd+A e Delete para apagar o conteúdo padrão (ex.: function myFunction() {}). Depois cole o código baixado em “Downloads” e salve.
    • Se o nome estiver Código.gs, você pode renomear para Code.gs (opcional, apenas para padronizar).
    • Se não existir: clique no botão “+” ao lado de Arquivos → Arquivo de script → digite apenas Code (sem extensão). O Apps Script criará Code.gs. Abra, apague o conteúdo e cole o nosso código. Salve.
  2. index.html (interface)
    • Se já existir index.html: abra, pressione Ctrl+A/Cmd+A e Delete para apagar o conteúdo padrão. Depois cole o código baixado em “Downloads” e salve.
    • Se não existir: clique no “+” → HTML → digite apenas index (sem extensão). O Apps Script criará index.html. Cole o nosso código e salve.
  3. Como abrir os arquivos baixados
    Abra Code.gs e index.html em um editor simples de texto (Windows: Bloco de Notas; macOS: TextEdit em Formato > Converter em Texto Simples). Se algum arquivo vier como .txt, apenas abra, copie o conteúdo e siga os passos acima. Evite Word/Google Docs para não alterar aspas e formatação.

3. Permissões (primeira execução)

  1. No editor, selecione a função semearLogoUmaVez.
  2. Clique em Executar (▶).
  3. Na primeira execução, autorize as permissões: Revisar permissões → escolha a conta → Avançado → Ir para (não seguro) → Permitir.

4. Publicar como Aplicação Web

  1. Implantar → Nova implantação → Tipo: Aplicação da Web.
  2. Executar como: Eu (sua conta).
  3. Quem tem acesso: Qualquer pessoa.
  4. Clique em Implantar e copie a URL gerada.
URL GERADA: https://script.google.com/macros/s/[ID]/exec

MANUTENÇÃO E SUPORTE

Funções administrativas (opcional)

  • aplicarLayoutEmTodas() — atualiza visual de planilhas.
  • aplicarLogoEmTodas() — aplica logo em massa.
  • hotfixResumoSemFormulaEmTodas() — corrige cálculos antigos.

CHECKLIST FINAL

Para a Equipe (antes de entregar):

  • Implantação publicada e URL testada.
  • Planilha sendo criada no Drive.
  • Interface funcionando no celular.
  • Links de download (Code.gs / index.html) funcionando.

Para o Empreendedor (orientações):

  • Sempre usar a mesma conta Google.
  • Autorizar acesso ao Drive no primeiro uso.
  • Lançar movimentações na data correta.
  • Backup automático no Drive (já habilitado).
